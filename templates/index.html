About SENTIVISION

Thesis Title

SentiVision: A Web-Based System for Real-Time Customer Emotion and Sentiment Analysis Using CCTV Camera Feeds in Pila, Laguna Restaurants

Project Overview

SentiVision is an AI-powered emotion analysis system designed to help restaurant owners in Pila, Laguna understand their customers' emotions and sentiments in real-time. By analyzing CCTV camera feeds, the system provides insights into customer satisfaction, enabling businesses to improve service quality and customer experience.

Key Features

  • Real-Time Analysis: Processes CCTV feeds with minimal latency
  • Seven Emotion Detection: Happy, Sad, Angry, Fear, Surprise, Disgust, Neutral
  • Binary Sentiment Classification: Positive and Negative grouping
  • Multi-Person Tracking: Persistent customer IDs via YOLO tracking
  • Comprehensive Reports: Detailed analytics, per-person breakdown, PDF export
  • Database Storage: SQLite with WAL mode for concurrent access
  • Visual Dashboard: Interactive charts for easy interpretation
